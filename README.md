# Contest-1
This is the github repo for contest 1 - Comp Arch. This team includes Angelina Martis and Arya Patel.

How to compile:
Use Visual Studio with the Irvine libraries on your machine.
Go to the Invisiball folder and find test1.asm, this is the only code needed for this game.
Download and move test1.asm into the Irvine folder on your machine.
Then put it into your project, build, and run without debuging.

If you happen to get a pop-up while debugging saying it contains a "virus", simply delete line 2 (includelib Irvine32.lib).
