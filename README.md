# Contest-1
This is the github repo for contest 1 - Comp Arch. This team includes Angelina Martis and Arya Patel.

# How to compile:<br>
Use Visual Studio with the Irvine libraries on your machine.<br>
Go to the Invisiball folder and find test1.asm, this is the only code needed for this game.<br>
Download and move test1.asm into the Irvine folder on your machine.<br>
Then put it into your project, build, and run without debuging.<br>

# "Virus"
If you happen to get a pop-up while debugging saying it contains a "virus", simply delete line 2 (includelib Irvine32.lib).
